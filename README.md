## Note:
1. Saat memulai, selalu lakukan {git pull} untuk menarik file dengan perubahan terbaru dari Server GitHub ke Lokal
2. Setelah selesai edit, jangan lupa Push ke GitHub
- {git add .} (mengumpulkan semua file untuk di-push)
- {git commit -m "Pesan-Up"} (isikan Pesan-Up dengan perubahaan yang dilakukan)
- {git push} (menyimpan perubahan ke GitHub)
3. Isi/Update Log agar perubahan bisa di-Tracking setiap melakukan perubahan

## Run Aplikasi di HP:
1. Pastikan Developer Options di HP sudah dinyalakan
2. Colokan Kabel USB dari laptop dan hp
3. Akan ada permintaan, "Allow USB Debugging?" pilih Allow
4. Buka file main.dart, lakukan Run/Run Without Debug
5. Akan muncul notifikasi di hp "Install App?" pilih Instal
6. Aplikasi akan terbuka
7. Setiap melakukan perubahan di coding, tinggal save dan aplikasi di hp akan terupdate tanpa harun Run ulang (fitur HotReload)
- Pastikan sambungan aman dan jangan terputus, jika terputus harus Run ulang
- Setelah install di hp, aplikasi sudah Standalone, tidak memerlukan tersambung di laptop. namun harus dicolok jika ingin melakukan perubahan
